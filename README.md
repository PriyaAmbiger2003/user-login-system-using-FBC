Django User Login System (Function-Based Views)

This is a simple User Authentication System built with Django using Function-Based Views (FBVs). The project demonstrates how to handle user registration, login, logout, and session management in a clean and easy-to-understand way.

Features

User Registration: Allow new users to create an account.

User Login: Authenticate existing users using username and password.

User Logout: End user sessions securely.

Function-Based Views: All functionality is implemented using Django's FBVs.

Session Management: Secure session handling for logged-in users.

Basic Validation: Form validation for registration and login.

Technologies

Django

Python

SQLite (default database)

HTML/CSS for templates

How to Run

Clone the repository:

git clone <repo-url>


Navigate to the project folder:

cd project_name


Install dependencies:

pip install -r requirements.txt


Apply migrations:

python manage.py migrate


Run the development server:

python manage.py runserver


Open your browser at http://127.0.0.1:8000/
